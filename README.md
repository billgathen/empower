# Empower

Go + React app that helps you accomplish your goals by:
- Organizing them and keeping your highest priorities in sight
- Brainstorming steps you can take to reach them
- Tracking your progress toward success

## To run locally

- Install Go v1.25.6+ [https://go.dev/doc/install](https://go.dev/doc/install)
- ```git clone https://github.com/billgathen/empower```
- ```cd empower```
- ```make dev```
- In your browser, open http://localhost:3000

## To develop locally

- Follow all steps to run locally: hot-reloading of Go/html/css is enabled
- Open another terminal window on ```empower``` folder
- Install Node v25.2.1+ [https://nodejs.org/en/download](https://nodejs.org/en/download)
- ```cd web```
- ```npm install```
- ```npm run watch``` enables hot-reloading of React
